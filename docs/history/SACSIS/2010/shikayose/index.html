SACSIS2010特別企画：「鹿寄せ」のご案内

概要

奈良ならではの企画として、２日目朝に「鹿寄せ」を企画しました。会議の前にささやかな癒しを提供します。
（提供　株式会社アックス様）

日時・場所

SACSIS2010 2日目 ５月２８日 朝９時（約20分）雨天決行
奈良市春日野町　奈良公園・飛火野


鹿せんべい

十分な数の鹿せんべいを準備します。心ゆくまで鹿とお戯れください。
またSACSIS期間中（初日を含む）は、受付に可能な限り鹿せんべいを準備いたします。外出時にご自由にご利用ください。

注：鹿せんべいは米ぬかと雑穀で出来ており、人間にとっても害はありませんが、食品ではありませんので食べないでください。


注意事項

  • 鹿せんべいを高く上げると鹿がお辞儀をしておねだりをします。じらしすぎると、頭突き等で攻撃してきます。
  • 鹿せんべいが無くなってもおねだりされることがあります。そのときは両手をパーにしてもう持っていないことをアピールしてください。
  • ポケットなどに隠し持っていると匂いでばれることがあります。
  • 隠し持っていなくても、鞄の中を探してくることがあります。持ち物には注意してください。


アクセス

「近鉄奈良駅バス停１番乗り場」 または 「JR奈良駅バス停１番乗り場」より 「系統番号２：市内循環（外回り）」に乗車。
運賃は乗車時前払い200円。CI-CA、ICOCA、PiTaPaが使えます。それ以外のICカード（Suica、PASMO等）は使えません。
春日大社表参道バス停下車（大仏殿春日大社前バス停の次） すぐ。
鹿寄せ終了後、会場の新公会堂までは徒歩約５分。

※市内循環（内回り）も逆ルートで運行しています。市内のホテルに宿泊されている方はこちらもご利用ください。（この場合、鹿寄せ会場は道路を渡った反対側になります。）

徒歩では近鉄奈良駅から約２５分。JR奈良駅から約３０分程度です。春日大社を目指して歩き、表参道の右側です。参考：春日大社との位置関係。
